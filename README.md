# Breast Cancer Histopathology Classification -- BreaKHis Analysis Network

A lightweight tool for classifying breast histopathology images into **Benign (low risk)** vs **Malignant (high risk)** using a **ResNet-based** deep learning model.

Dataset: BreaKHis 400X (Kaggle)  
https://www.kaggle.com/datasets/forderation/breakhis-400x/data

---

## Project Structure
- **GUI/**: runnable interface for image-level prediction with tutorial.
- **model/**: stores training outputs and the **best trained checkpoint (`.pth`)**.
